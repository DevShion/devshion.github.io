## Support for Apps
If you have any questions, bug reports, or feature requests regarding our apps, please use one of the following methods:

当方が提供するアプリに関するご質問、バグ報告、機能要望については、以下のいずれかの方法でご連絡ください。

### 1. GitHub Issues (Recommended / 推奨)
Please create a new issue in this repository.
本リポジトリの **[Issues](https://github.com/DevShion/devshion.github.io/issues)** より新規投稿をお願いします。

- Please include the **App Name** and **OS version**.
- アプリ名とOSのバージョンを記載してください。

### 2. Email Support / メールでのお問い合わせ
If you do not have a GitHub account, please contact us via email.  
GitHubアカウントをお持ちでない場合は、以下のメールアドレスまでご連絡ください。

**Email:** `shionpro.developer@gmail.com`
